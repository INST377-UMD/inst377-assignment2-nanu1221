In this web page, there are options to navigate using Audio!

Ensure your mic is enabled to be able to use this feature.

  • Navigate to a different page: Say: Navigate to <page>
  • Change the Page Color: Say: Change the color to <color>
  • Say Hello: Say: Hello Bato you!

Audio Instructions:

  • Hello: Says "Hello World" in an alert.
  • Change the color to [color]: Changes the background color of the page.
  • Navigate to [page]: Navigates to the respective page (Home, Stocks, Dogs).
  • Lookup [stock]: Shows the chart for the given stock ticker (default 30 days).
